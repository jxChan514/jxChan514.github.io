Research Experience

National Key Project: “Hexapod Mobile Robot with Physical-Environment Perception”
Core Member 2024.05 - 2025.06

Developed a physically-aware hexapod robot for embodied intelligence in complex environments. Built a distributed computing-control architecture and designed a hardware time-aligned multi-sensor fusion chain, reducing temporal jitter among vision, inertial, and tactile signals. As hardware lead, I developed a highly integrated control system with depth camera, IMU, and pressure array, and optimized power and high-bandwidth communication for lower sensing-action loop latency. The platform achieved stable data transmission on unstructured terrain, supporting high-frequency SLAM and YOLO deployment.

Target Detection Based on FMCW Radar
First Author / Project Lead 2024.09 - 2025.08

Built a complete mmWave radar perception pipeline for behavior monitoring in complex indoor scenarios. Combined 2D-FFT with micro-Doppler feature extraction, and introduced dynamic-threshold CFAR plus cross-domain temporal analysis for weak-target detection in non-stationary signals. Led the full pipeline from raw I/Q signals to time-frequency feature engineering, and optimized MTI to suppress static clutter. The project ranked 1st (98/100), demonstrating strong robustness for real-time behavior recognition.

Multimodal LLM (vLLM) RL Alignment and Inference Optimization
